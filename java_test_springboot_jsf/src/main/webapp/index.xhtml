This file illustrates PrimeFaces bug 6184 (see https://code.google.com/p/pfaces/issues/detail?id=6184).

After clicking the button there should be two error messages behind the input field.
  • Using the Client Side Validation, both error messages are displayed behind the input field.
  • Using Server Side Validation, only one message is displayed. Unfortunately, the second message is an error message, but it is suppressed.
To activate server side validation, you have to set the parameter in de.beyondjava.main.Initializer (line 15) and you have to remove the annotations of de.beyondjava.jsf.sample.searchExpressions.SmallNumberBean (lines 29 and 30).
